## Linux常用命令总结

对Linux上一些比较常用但又难记或者参数较多的命令作一个总结。

### 进程、端口相关

```shell
//查看端口使用情况
netstat -anp | grep 端口号

//查看指定进程
ps -aux | grep 进程关键词

//查看任务管理器
top
```

### 搜索相关

```shell
//在文件中收索关键词上下10行，并高亮显示关键词
grep -C 10 "关键词" 文件名 --color

//找到当前目录及其子目录下占用空间最大的5个文件
du -h . | sort -rh | head -n 5

//按照各种规则查找文件
find 路径 -name "文件名"
find 路径 -type 类型
```

### 文件、文件夹相关

```shell
//递归删除文件夹
rm -r +文件夹名称

//压缩文件
zip 压缩后的文件名.zip 需要压缩的文件夹名

//解压文件
unzip file.zip  

//打包一个归档
tar -cvf out.tar file  

//释放一个归档
tar -xvf file.tar 
```

### 系统管理相关

```shell
//列出正在运行的 Unit(Systemd 可以管理所有系统资源)
systemctl list-units

//杀死一个服务的所有子进程
sudo systemctl kill apache.service

//立即启动一个服务
sudo systemctl start apache.service

//立即停止一个服务
sudo systemctl stop apache.service

//重启一个服务
sudo systemctl restart apache.service
```

### Java相关

```shell
//运行jar包，并将错误信息输入到error.log
nohup java -jar project-0.0.1-SNAPSHOT.jar  > error.file  2>&1 &

//反编译字节码文件，得到源代码(编译后的)
//Java执行流程: .java -> 编译后的源代码 -> .class -> JVM解释执行
javap -c -v Test.class

/*
查看java虚拟机状态信息,每250毫秒，获取5个采样数据.
S0: 第一块幸存者空间使用情况
S1: 第儿块幸存者空间使用情况
E : 伊甸园空间使用情况
O : 老年代空间使用情况
M : 元空间使用情况, JDK1.8
P ： 方法区(永久代)使用情况，JDK1.8以前
...
*/
jstat -gcutil Java进程id 250 5
```



### Docker相关

```shell
//查看所有本地主机上的镜像 
docker images
//搜索镜像
docker search
//下载镜像 
docker pull 镜像id
//删除镜像 
docker rmi  镜像id
//列出所有正在运行的容器(-a 查看所有容器)
docker ps
//查看容器中进程信息ps
docker top 容器id 
//进入容器内部
docker exec -it 容器id bash
//启动容器
docker start 容器id	
//重启容器
docker restart 容器id	
//停止当前正在运行的容器
docker stop 容器id	
//强制停止当前容器
docker kill 容器id	
```
